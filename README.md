# 📊 Inflation Dynamics Analysis

This project analyzes inflation trends in Pakistan using statistical and econometric techniques to identify key macroeconomic drivers and forecast future inflation.

---

## 📌 Overview

The study examines inflation behavior using historical data (1980–2023) and evaluates the impact of key economic variables such as money supply, exchange rate, GDP growth, food imports, and oil prices.

---

## 🎯 Objectives

- Analyze historical inflation trends  
- Identify key macroeconomic drivers of inflation  
- Apply statistical and time-series models  
- Compare model performance and forecasting ability  

---

## 📊 Data & Variables

Key variables used:
- Inflation (target variable)  
- Money Supply (M2)  
- Exchange Rate  
- GDP Growth  
- Food Imports  
- Crude Oil Prices  

---

## ⚙️ Methods Used

- Time Series Modeling (AR, MA, ARMA, ARIMA)  
- Regression Models:
  - Ridge Regression  
  - LASSO  
  - Elastic Net  
- Correlation and statistical analysis  

---

## 📈 Key Findings

- Exchange rate and oil prices show a positive relationship with inflation  
- GDP growth has a negative relationship with inflation  
- Money supply showed weak correlation in this dataset  
- ARIMA performed best among all models based on MSE
